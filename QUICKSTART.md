## 🎯 Шаги для запуска на BotHost:

1. **Зайдите на https://bothost.pro**

2. **Создайте новый проект:**
   - Войдите в аккаунт
   - Нажмите "Создать проект" или "New Bot"
   - Выберите Python 3.11

3. **Загрузите файлы через GitHub:**
   - Создайте репозиторий на GitHub
   - Загрузите все файлы из этой папки:
     - main.py
     - requirements.txt
     - Procfile
     - runtime.txt
     - README.md
   - Подключите репозиторий в BotHost

   **ИЛИ загрузите файлы напрямую:**
   - Перетащите файлы в интерфейс BotHost
   - Убедитесь что main.py в корне

4. **Запустите бота:**
   - Нажмите кнопку "Start" или "Deploy"
   - Подождите установки зависимостей (1-2 минуты)
   - Бот автоматически запустится

5. **Проверьте работу:**
   - Откройте Telegram
   - Найдите вашего бота
   - Отправьте /start
   - Вы должны получить приветственное сообщение

## 🔧 Альтернативные хостинги (если BotHost не подходит):

### Railway.app (Проще всего):
1. Зайдите на https://railway.app
2. Подключите GitHub репозиторий
3. Railway автоматически всё настроит
4. Бот запустится сам

### Render.com:
1. Зайдите на https://render.com
2. New > Background Worker
3. Подключите GitHub
4. Build Command: `pip install -r requirements.txt`
5. Start Command: `python main.py`

### PythonAnywhere:
1. Зайдите на https://www.pythonanywhere.com
2. Files > Upload files (main.py, requirements.txt)
3. Console > Bash:
   ```bash
   pip3 install --user -r requirements.txt
   ```
4. Tasks > Add new task:
   ```
   python3 /home/yourusername/main.py
   ```

### Replit:
1. Зайдите на https://replit.com
2. Create Repl > Python
3. Скопируйте содержимое main.py
4. Создайте requirements.txt
5. Нажмите Run
